📖 URDF Builder Usage Guide (click to expand) ▶

Basic Workflow

1. Add Joints: Click "Add Joint" to add articulations. View parent-child relationships in the Structure Map.

2. Edit Properties: Select a joint to edit its position (Kinematics), appearance (Visual), and collision area (Collision).

3. Base Settings: Click the blue Base Joint node in the Structure Map to edit base settings (Fixed/Floating, Visual, Collision).

4. Export: Set the robot name in the Export panel, optionally upload an Inertia file, and click "Generate URDF".

Key Settings

Axis / Type: Roll=X-axis, Pitch=Y-axis, Yaw=Z-axis, Fixed=no rotation

Offset XYZ: Position offset from parent link (in mm)

Limits Min/Max: Joint rotation limits in degrees

Inertia File Format

link_namemassixxiyyizzixyixziyzcom_xcom_ycom_z
base_link5.00.010.010.010.00.00.00.00.00.1

Row order: base_link → joint_0's child → joint_1's child ... Units: mass(kg), inertia(kg·m²), com(m)
